Weeks 1–3

Excel

The tool every analyst uses on day one. Nail the core formulas, pivots, and dashboards before anything else.

  • SUM, AVERAGE, COUNTIF, IF statements
  • Pivot tables
  • VLOOKUP & INDEX MATCH
  • Conditional formatting & basic dashboards

Weeks 3–8

SQL

The #1 skill in every data analyst job posting. Fundamentals first, then window functions that show up in every interview.

  • SELECT, WHERE, GROUP BY, JOINs
  • Subqueries & CTEs
  • ROW_NUMBER, RANK, LAG, LEAD
  • 75 practice questions + real datasets

Weeks 6–11

Python

How you scale your analysis beyond spreadsheets. Learn foundations, then move straight into Pandas — the library you'll use every single day.

  • Variables, loops, functions, conditionals
  • Jupyter Notebook setup
  • Pandas: read, filter, groupby, merge
  • matplotlib & seaborn basics

Weeks 9–13

Data Visualization

Pick one — Tableau or Power BI. Clean, story-driven dashboards beat flashy ones every time. Learn the one that's on more local job postings.

  • Connecting to data sources
  • Bar, line, area & scatter charts
  • Dashboard design principles
  • Filters, drill-downs & publishing

Weeks 10–13

Statistics Basics

You don't need a stats degree — just enough to understand what the numbers mean and draw the right conclusions from your data.

  • Mean, median, standard deviation
  • Correlation vs causation
  • Hypothesis testing & p-values
  • Normal distribution

Portfolio

2–3 projects on GitHub is enough. Take a messy real-world dataset, ask a business question, and deliver a clear answer with a recommendation.

  • Real datasets: e-commerce, HR, finance
  • End-to-end: question → insight → action
  • GitHub with clean READMEs
  • Problem · Approach · Key Findings

Technical Confidence
